Politique de Confidentialité

Dernière mise à jour : 30 Mai 2025

En Résumé

  • 🔒 Aucune donnée personnelle collectée
  • 📊 Analytics anonymes pour améliorer l'application
  • 📢 Publicités non-personnalisées via AdMob
  • ⚙️ Contrôle simple via les paramètres système

1. Données Collectées

1.1 Données Techniques (Firebase)

Données anonymisées essentielles :

  • 📱 Type d'appareil et version du système d'exploitation
  • ⏱️ Durée des sessions d'utilisation
  • 🛠️ Fonctionnalités utilisées (sans identification)
  • ⚠️ Rapports d'erreurs techniques (diagnostics)

1.2 Données Publicitaires (AdMob)

Strictement nécessaires pour les publicités :

  • 🆔 Identifiant publicitaire Google (AAID)
  • 🌍 Langue et pays de l'appareil
  • 📶 Type de connexion (WiFi/mobile)
  • Finalité : Financement de l'application via des pubs contextuelles
  • Partage : Données transmises à Google conformément à leurs règles de confidentialité
  • Opt-out : Désactivable dans les paramètres système Android/iOS

2. Utilisation des Données

Ces données servent exclusivement à :

  • 🛠️ Améliorer les fonctionnalités de l'application
  • 📈 Comprendre l'usage global (sans tracking individuel)
  • 🆓 Maintenir le service gratuit via les publicités
  • 🐞 Corriger les bugs et problèmes techniques

Les données analytiques sont activées par défaut pour améliorer l'app.

3. Protection des Données

Nos engagements :

  • 🚫 Aucun partage avec des tiers non-autorisés
  • 🔐 Données techniques anonymisées chez Firebase
  • 📜 Permissions Android minimales et justifiées

4. Confidentialité

Nous n'avons aucun accès à vos données personnelles. Les outils tiers (Google) respectent leurs propres politiques.

5. Vos Contrôles

Vous pouvez :

  • 📢 Gérer les publicités via les paramètres Google
  • 🔄 Réinitialiser votre identifiant publicitaire (dans les paramètres système)

6. Conformité Juridique

Nous respectons :

  • 🇪🇺 Le RGPD (Règlement Général sur la Protection des Données)
  • 📱 Les directives du Google Play Store
