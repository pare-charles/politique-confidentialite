Politique de Confidentialité

Dernière mise à jour : 20 Mai 2025

1. Responsable

Je suis un particulier développant l'application Yaam Gestion Pro depuis le Burkina Faso.

2. Données Collectées

L'application utilise Firebase Analytics (service de Google) pour :

  • Analyser l'usage (ex : durée des sessions, fonctionnalités utilisées).
  • Détecter les erreurs techniques.

Aucune donnée personnelle (nom, email, etc.) n'est collectée.

3. Partage des Données

Les données anonymes sont traitées par :

  • Google Firebase (serveurs aux États-Unis).

4. Sécurité

Les données sont chiffrées et anonymisées conformément aux standards de Google.

5. Vos Droits

Vous pouvez :

  • Désactiver Firebase Analytics dans les paramètres de l'app.
  • Me contacter pour toute question : yaamgestionpro@gmail.com.
